# Recipe Dashboard

Dieses Projekt ist eine kleine Svelte-App, die mit Vite gebaut wurde.

## Voraussetzungen

- **Node.js** (empfohlen: aktuelle LTS-Version, z. B. 20+)
- **npm** (wird mit Node.js installiert)

## Installation

1. Repository klonen oder herunterladen.
2. In den Projektordner wechseln:

   ```bash
   cd cookbook
   ```

3. Abhängigkeiten installieren:

   ```bash
   npm install
   ```

## App lokal nutzen (Entwicklungsmodus)

Starte den lokalen Entwicklungsserver:

```bash
npm run dev
```

Danach findest du die App in der Regel unter:

- <http://localhost:5173>

Der Server aktualisiert sich automatisch bei Änderungen am Code.

## Produktions-Build erstellen

```bash
npm run build
```

Die fertigen Dateien landen im Ordner `dist/`.

## Produktions-Build lokal testen

```bash
npm run preview
```

Damit kannst du den gebauten Stand lokal prüfen.
